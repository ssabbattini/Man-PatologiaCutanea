# Dashboard

## Workflow

La stesura di una diagnosi, dall'accettazione alla pubblicazione sul web di un referto si compone dei seguenti passaggi:

* Accettazione
* Scrittura diagnosi
* Correzione diagnosi
* Pubblicazione su web del referto
